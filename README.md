## About the course

This course is for developers to create end-to-end solutions in Microsoft Azure using:
- Azure App Services
- Azure Compute solutions
- Azure Functions
- Azure storage
- Authentication and authorization
- Security using Azure KeyVault and Managed Identities
 
<br/>
